# Primer Buyer Personas

Twenty personas representing the people most likely to actively search for a product like Primer — a $5,000 perpetual-license, self-hosted performance management platform that ships as SvelteKit source code.

Each persona includes role, industry context, core pain, example search queries, and the specific hook(s) in Primer's value proposition that convert them.

---

## 1. "Frustrated-with-Workday" HR Director — Dana

- **Role:** Director of HR, 400-person industrial manufacturer (Ohio)
- **Pain:** Paying $85K/yr for a performance module nobody uses; managers refuse to open it
- **Searches:** _"alternative to workday performance"_, _"simple KPI scorecard self-hosted"_
- **Trigger:** Renewal quote just landed; CFO asked "what do we actually get for this?"
- **Hook:** TCO page, perpetual license, 5-tier framework simplicity

## 2. Data-Sovereignty CIO — Marcus

- **Role:** CIO at a regional credit union (~600 employees)
- **Pain:** Compliance team banned new SaaS vendors handling employee data
- **Searches:** _"self-hosted performance management on-prem"_, _"OKR tool no cloud"_
- **Values:** Source code ownership, no external API calls, auditable code
- **Hook:** Security page, deployment page, "your infrastructure vs DavidPM, LLC" diagram

## 3. Private-Equity Operating Partner — Sasha

- **Role:** Operating Partner, lower-middle-market PE firm
- **Pain:** Needs a standard scorecard she can deploy across 12 portfolio companies without a new SaaS contract per portco
- **Searches:** _"portfolio company KPI dashboard"_, _"perpetual license performance tool"_
- **Buying lever:** One $5K license = instant rollout per portco
- **Hook:** Price, deployment flexibility

## 4. Bootstrapped SaaS Founder/CEO — Jin

- **Role:** CEO, 35-person bootstrapped B2B SaaS
- **Pain:** Team doesn't agree on what "good" looks like for any function
- **Searches:** _"how to set KPIs for small company"_, _"OKR tool open source"_
- **Objection:** "Can I just use a spreadsheet?" — converted by the demo's weighted composite score
- **Hook:** Interactive demo, affordable one-time price

## 5. Chief of Staff — Priya

- **Role:** CoS to CEO, 250-person Series C health-tech
- **Pain:** Owns QBRs; metrics stale, every department reports a different "truth"
- **Searches:** _"cross-functional KPI alignment tool"_, _"strategy execution dashboard"_
- **Quote that lands:** "82% of middle managers feel invisible"
- **Hook:** Problem page research, inquiry mechanism, goal cascading

## 6. Construction Project Director — Big Ed

- **Role:** Project Director, commercial GC (200 field + office staff)
- **Pain:** Schedule adherence + safety + budget all tracked in separate systems
- **Searches:** _"construction KPI scorecard"_, _"schedule adherence dashboard"_
- **Resonance:** Demo uses construction metrics (Schedule Adherence, Safety Record) as the default example
- **Hook:** Industry-specific demo, Meridian Construction seed data

## 7. Hospital Quality & Ops Director — Dr. Reyes

- **Role:** VP Quality, 600-bed regional hospital
- **Pain:** PHI-adjacent data can't leave the network; needs tier-based metric governance
- **Searches:** _"HIPAA-compliant performance management"_, _"on-prem OKR healthcare"_
- **Won by:** Self-hosted architecture + SQLite/Postgres flexibility
- **Hook:** Security page, deployment flexibility, data-residency guarantees

## 8. Professional Services Managing Partner — Claire

- **Role:** Managing Partner, 80-person consultancy
- **Pain:** Utilization + win-rate + client-NPS all matter but aren't weighted; review cycles are political
- **Searches:** _"partner scorecard consulting firm"_, _"utilization weighting tool"_
- **Hook:** Weighted composite score, transparent waterfall, inquiry mechanism for political conflicts

## 9. Government / Public Sector PMO Lead — Terrence

- **Role:** PMO Director, state transportation agency
- **Pain:** Procurement blocks SaaS; needs source code + perpetual license for 5-year budget cycle
- **Searches:** _"government performance management source code"_, _"GSA-compatible OKR"_
- **Buying lever:** Perpetual license fits capital-expense accounting
- **Hook:** Source code delivery, perpetual license, no vendor lock-in

## 10. International HR Lead (Multilingual Org) — Amélie

- **Pain:** Existing tools English-only; Arabic RTL especially broken
- **Searches:** _"multilingual performance review system"_, _"RTL HR tool Arabic"_

- **Hook:** i18n across 9 languages including Arabic RTL

## 11. EOS / Traction Implementer — Rick

- **Role:** Certified EOS Implementer coaching ~15 client companies
- **Pain:** Clients' scorecards live in Google Sheets and rot fast
- **Searches:** _"EOS scorecard software"_, _"weekly scorecard tool"_
- **Sell:** One license → clone + rebrand per client (source code)

## 12. IT Buyer Escaping Vendor Lock-in — Omar

- **Role:** Director of IT, 180-person law firm
- **Pain:** Just got burned by a vendor sunsetting a module mid-contract
- **Searches:** _"perpetual license performance software"_, _"no subscription OKR"_
- **Values:** "$5K once, mine forever"
- **Hook:** Perpetual license, source code ownership, no vendor dependency

## 13. Retail Regional Director — Teresa

## 14. Franchise System Ops Leader — Wade

- **Pain:** Needs a system franchisees will actually open; per-seat SaaS is cost-prohibitive
- **Searches:** _"franchise performance dashboard"_, _"multi-unit KPI tool no per-seat fee"_

## 15. Manufacturing Plant Manager — Heinrich

- **Pain:** Works councils reject cloud employee-monitoring tools; GDPR-sensitive
- **Searches:** _"GDPR performance management self-hosted"_, _"Werksleiter KPI Tool"_
- **Fit:** EU data residency + source code audit satisfies works council
- **Hook:** Self-hosted, German locale, GDPR-compatible architecture

## 16. Army of One: Head of People — Nadia

- **Role:** First People hire, 120-person fintech
- **Pain:** CEO wants a performance system by Q3; no budget for enterprise SaaS
- **Searches:** _"affordable performance management startup"_, _"cheap OKR platform"_
- **Price reaction:** $5K lands flat vs. $30K/yr Lattice quote
- **Hook:** Price, simplicity, fast time-to-value

## 17. Operations Consultant — David

- **Role:** Independent ops consultant, former COO
- **Pain:** Wants a standard framework he deploys for clients as part of engagements
- **Searches:** _"white label OKR software"_, _"rebrandable KPI dashboard"_

- **Hook:** Source code access, rebrand capability

## 18. Defense / Cleared Environment Engineer — Colonel Ward (Ret.)

- **Role:** IT Director, defense contractor
- **Pain:** Needs a performance tool that can run inside an air-gapped network
- **Searches:** _"air gapped performance management"_, _"offline OKR tool"_
- **Fit:** No external dependencies, SQLite mode for small deployments
- **Hook:** Air-gap deployment, no phone-home, SQLite option

## 19. Skeptical VP of Engineering — Ravi

- **Role:** VP Eng, 90-person SaaS
- **Pain:** Hates HR-driven review tools; wants engineering-owned metrics (DORA, reliability)
- **Searches:** _"engineering performance metrics platform"_, _"team owned KPI tool"_

## 20. Nonprofit Executive Director — Sister Grace

- **Role:** ED, mid-size nonprofit (education, 140 staff + volunteers)
- **Pain:** Board wants program-outcome scorecards; can't justify $20K/yr SaaS
- **Searches:** _"nonprofit outcomes dashboard"_, _"program KPI tracking affordable"_
- **Fit:** One-time $5K fits grant-funded capital buy
- **Hook:** Price, perpetual license, capital-expense compatibility
